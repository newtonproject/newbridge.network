    NewBridge API 文档 #

    欢迎使用面向开发人员的 NewBridge API 文档。

    简介 #

    加入正在构建下一代跨链服务的开发人员的行列。 我们全面的文档和指南可帮助您探索 NewBridge API 和功能，并尽快集成以使用 NewBridge。

    前端 APIs #

    API Endpoints

    API Endpoints for NewBridge

    (Core) Pairs Info API

    查找不同桥接方向的链和资产的桥接对

    (Core) Account API

    查找用户地址的存款地址

    (Core) Account History API

    返回一个JSON结构，其中包含指定帐户的NewBridge桥接历史记录的详细信息

    (XChain) 创建账户

    NewBridge 的 API Endpoints

    XChain API #

    Chain API 是运行在 main api 及 newbridge core后端的程序，其接口以 grpc 定义。

    ChainAPI 服务定义如下：

    // The ChainAPI service definition.
service ChainAPI {
    rpc CreateAccount (CreateAccountRequest) returns (CreateAccountReply) {}
}
